Welcome to the Bookstore API!

This is a simple Express.js application with MongoDB and TypeScript.

Available API Endpoints:

  • GET /api/books - Get all books
  • GET /api/books/:id - Get a book by ID
  • POST /api/books - Create a new book (Body: { "title": "...", "author": "...", "isbn": "..." })
  • PUT /api/books/:id - Update a book by ID (Body: { "title": "...", "author": "...", "isbn": "..." })
  • DELETE /api/books/:id - Delete a book by ID
  • GET / - Serve this HTML file
